Transcript support

Anytime Podcast Player supports Podcasting 2.0 transcripts, allowing you to search or follow along to a podcast episode.

Transcripts are provided by the podcast host.

Why is there no transcript for this episode?

There are several reasons why Anytime may be unable to display a transcript for the episode:

  • The host's provider does not support Podcasting 2.0 transcripts.
  • The host has not generated a transcript for this episode.
  • The host has not generated a transcript for this episode that Anytime can support (currently SRT and JSON formats).
  • The host has released a transcript since Anytime last checked for updates.
